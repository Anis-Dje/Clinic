# HealthCare Clinic Website

A professional, fully responsive medical clinic website built with HTML5, CSS3, JavaScript, and jQuery.

## Features

### Pages
- **Home (index.html)** - Welcome page with hero carousel, clinic information, and testimonials
- **About Us (about.html)** - Clinic history, mission, values, and meet our doctors section
- **Services (services.html)** - Medical services with interactive accordion
- **Appointment (appointment.html)** - Appointment booking form with validation
- **Contact (contact.html)** - Contact information, hours, map, and contact form

### Technologies Used
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Interactivity**: JavaScript (Vanilla & jQuery)
- **Responsive**: Mobile-first design approach
- **Animations**: CSS transitions and jQuery effects

### Key Features

#### JavaScript Functionalities
1. **Image Carousel** - Bootstrap carousel on home page showcasing clinic highlights
2. **Form Validation** - Comprehensive validation for:
   - Email format validation
   - Phone number format (10 digits)
   - Required field validation
   - Age calculation from birthdate
   - File upload validation (PDF, JPG, PNG - Max 5MB)

#### jQuery Functionalities
1. **Hover Effects** - Smooth animations on:
   - Navigation links
   - Feature cards
   - Doctor cards
   - Buttons
   - Testimonials
   
2. **Dynamic Form Inputs**
   - Time slot display/hide based on service selection
   - Dynamic doctor list updates based on selected service
   - Phone number auto-formatting
   - Real-time age calculation

3. **Accordion** - Medical services page with smooth expand/collapse animations

4. **Additional Interactions**
   - Smooth scrolling for anchor links
   - Navbar background change on scroll
   - Scroll animations for cards
   - Active page indicator in navigation
   - Form focus effects

### Form Features

#### Appointment Form
- Personal information (name, email, phone, address)
- Birthdate with age calculation
- Gender selection
- Service selection with dynamic doctor updates
- Preferred date and optional time slot
- Medical history/allergies textarea
- Medical file upload

#### Contact Form
- Name, email, phone fields
- Subject and message
- Email validation
- Phone number formatting

### Responsive Design
- Mobile-first approach
- Bootstrap 5 grid system
- Responsive navigation with hamburger menu
- Optimized layout for all screen sizes

### Installation & Usage

1. **Download the project files**
2. **Open in a web browser** - Simply open `index.html` in your browser
3. **No build process required** - All files are ready to use
